## 1. Клонирование репозитория

git clone <https://github.com/boronin02/cyberia-practice.git>
cd cyberia-studio/frontend

## 2. Переменные окружения

cp .env.example .env

## 3. Установка зависимостей

yarn install

## 4. Сборка проекта

yarn build

## 5. Запуск проекта в режиме разработки

yarn dev

## 6. Запуск собранного приложения

yarn start
